<Section icon={Map} title="ステップバイステップ：ガレージカスタマイズ＆保存ガイド">

カスタムビルドを作成するには、ガレージインターフェースを操作し、保存システムを利用する必要があります。未保存の改造は、車両がデスポーンされたりサーバーを離れたりすると失われます。カスタムビルドを永久に保存するために、以下のステップバイステップのプロセスに従ってください。

<Steps>
  <Step num="1" title="ガレージインターフェースにアクセス">
    車両を指定されたガレージゾーンに運転して入れるか、プレイヤーHUDからカスタマイズメニューを開いて編集を開始します。
  </Step>
  <Step num="2" title="塗装と素材を選択">
    素材タブに移動し、プライマリおよびセカンダリのペイント仕上げを適用します。マット、メタリック、グロス、またはクロームから選択します。
  </Step>
  <Step num="3" title="カスタムリムを装備">
    リムメニューをブラウズし、車両クラスに合ったデザインを選択します。スタンスがドラッグレースの好みに合っていることを確認してください。
  </Step>
  <Step num="4" title="カスタムプリセットを保存">
    ガレージインターフェース内の保存メニューを開き、空のスロットを選択して、改造を確定するためにプリセットを保存します。
  </Step>
</Steps>

保存したプリセットを管理することで、ドラッグレース、ロールプレイ、警察の追跡など、状況に応じて異なる設定を素早く切り替えることができます。

| プリセット機能 | メニューコマンド | 実用的な用途 |
| :--- | :--- | :--- |
| **プリセット保存** | Save Build | 現在の塗装、リム、素材設定を保存 |
| **プリセット読み込み** | Load Build | 保存したカスタム車両を即座にスポーン |
| **車両リセット** | Clear Customization | 車両を工場出荷時の設定に戻す |

<Callout type="warning" title="保存に関する警告">
リセットボタンをクリックしたり、別の車両クラスをスポーンしたりする前に、必ずプリセットを保存したか再確認してください。未保存の改造は復元できません。
</Callout>

</Section>

<Section icon={Target} title="プロップスポナーとドラッグモードの設定">

Drag Drive Simulatorのガレージで作成したビルドをテストするために、新しく追加された「保存、読み込み、ドラッグモードプロップスポナー」を使用できます。このシステムにより、プレイヤーはマップ上のどこにでもカスタムドラッグストリップやステージングエリアを構築でき、組織的なコミュニティレースを促進できます。

適切なドラッグストリップを設置するには、ステージングプロップを配置し、公正なスタートを確保するためにプライベートサーバーコマンドを利用する必要があります。

| ツール / コマンド | アクション | 使用シナリオ |
| :--- | :--- | :--- |
| **プロップスポナーメニュー** | トラックバリアをスポーン | カスタムドラッグストリップの境界線を設定 |
| **ドラッグモードプロップ** | ステージングライトを配置 | スタートラインで車両を整列させる |
| **!cd [x]** | カウントダウンを開始 | 全ドライバーのレーススタートを同期 |

ドラッグレースを主催する際は、チャットで `!cd 5` と入力してカウントダウンコマンドを使用し、画面上に5秒間の視覚的なカウントダウンを表示させます。これにより、スタートラインでの混乱がなくなり、レースの結果が通信遅延ではなく、車両のチューニングと反応速度によって決まるようになります。

<Callout type="info" title="コマンド情報">
`!cd` コマンドは、変数を希望の秒数に置き換えることで任意の時間に調整でき、さまざまなレース形式に合わせて高度にカスタマイズ可能です。
</Callout>

</Section>
