<Section icon={Trophy} title="レーストラックとテスト場">

このゲームの心臓部はトラックにあります。この **Drag Drive Simulator マップチュートリアル**では、競技志向のプレイヤーにとって最も重要な2つの場所、**セントゥール・サーキット（Sentul Circuit）** と **ペッタラニ・トラック（Pettarani Track）** を紹介します。これらのエリアは、チューニング設定を検証し、競争的なレースを通じて大量のRPを稼ぐ場所です。

<Steps>
  <Step num="1" title="セントゥール・サーキットを特定する">
    メインスポーンから、郊外に向かう高速道路の標識に従ってください。セントゥールはドラッグレースの主要ハブであり、通常どのサーバーでも最も人が集まるエリアです。
  </Step>
  <Step num="2" title="マカッサル拡張エリアに入る">
    連絡トンネルを通ってマカッサル市に移動します。このエリアには、ロードレースとドラッグレースの両方のシステムを備えた新しいペッタラニ・トラックがあります。
  </Step>
  <Step num="3" title="レースを開始する">
    スタートラインに並びます。タイヤが温まっているか（該当する場合）、ギア比が特定のトラックの長さに合わせて設定されているか確認してください。
  </Step>
  <Step num="4" title="パフォーマンスを確認する">
    レース後、フィニッシュタイムを確認します。最高速度に達するのが早すぎる場合は、チューニングメニューに戻ってギアを長く設定し直してください。
  </Step>
</Steps>

トラックによって異なる戦略が必要です。セントゥールはクラシックなドラッグストリップで、ローンチコントロールと反応速度が最も重要です。一方、ペッタラニはより多様性があり、生の加速力に加えて車両のハンドリングやサスペンション設定をテストするロードスタイルのレースが可能です。

| トラック名 | 場所 | レースの種類 | 用途 |
| :--- | :--- | :--- | :--- |
| **セントゥール・サーキット** | オリジナルマップ | ドラッグレース | チューニング検証、RP稼ぎ |
| **ペッタラニ・トラック** | マカッサル市 | ドラッグ＆ロードレース | 高度なテスト、ハイステークス |
| **都市の街頭** | スラカルタ | ストリートレース | カジュアルなクルーズ、ロールプレイ |

<Callout type="success" title="プロのテストヒント">
正式なレースに参加せずに、トラック近くのフリーロームエリアを使用してローンチテストを行ってください。これにより、勝敗記録に影響を与えたり待ち時間で無駄にしたりすることなく、**ギア比**を微調整できます。
</Callout>

</Section>

**バリスタの仕事**は、現在最も人気のある稼ぎ方の一つです。オリジナルマップのコーヒーショップにあり、進行システムが特徴です。「ジュニア・バリスタ」（5k XP）や「シニア・バリスタ」（25k XP）などのバッジを獲得でき、それぞれに大きな現金ボーナスが付与されます。移動しながら稼ぎたいプレイヤーには、指定されたさまざまな場所に荷物を届けることでマップのレイアウトを覚えられる **運び屋の仕事** が適しています。

| 職種 | マップ上の場所 | 収入タイプ | 条件 |
| :--- | :--- | :--- | :--- |
| **バリスタ** | コーヒーショップ（オリジナル） | シフト制 / XP | 無料でプレイ可能 |
| **警察** | 警察署（各所） | パトロール / 奉仕 | Police Pass (149 Robux) |
| **運び屋** | 配達ハブ（オリジナル） | 配達ごと | 信頼できる車両 |
| **オフィスワーカー** | オフィスタワー（都市部） | タスク制 | 低負荷、安定した収入 |

<Callout type="info" title="仕事の効率化">
**2x Paycheck Pass** (499 Robux) を持っている場合、これらすべての場所からの収入が2倍になります。これは、マップ上のディーラーの在庫を素早く買い占めるための最も効率的な方法です。
</Callout>

<Section icon={Sparkles} title="チューニングハブとカスタマイズショップ">

愛車を改造する場所を知らなければ、マップの知識は完全ではありません。チューニングはこのゲームの「秘伝のソース」です。ストック状態のパフォーマンスでセントゥールで勝つことは稀です。カスタマイズハブを訪れて、ギア比、サスペンション、外観を調整する必要があります。

チューニングシステムはほとんどのエリアからアクセス可能ですが、特定の「スタンス」ショップや「ペイント」ショップではより深いカスタマイズが可能です。例えば、**Advance Paint Pass** ではHEXカラーのサポートやカーボンファイバー素材が使用可能になり、**Suspension Pro Pass** ではダンピングや剛性をコントロールできます。これは、ペッタラニ・トラックでハイパワーバイクを安定させるために不可欠です。

| チューニングカテゴリー | アクセス方法 | レースへの影響 | 推奨設定 |
| :--- | :--- | :--- | :--- |
| **ギア比** | メニュー / フリーローム | 加速 vs 最高速度 | ドラッグは短め、ロードは長め |
| **サスペンション** | カスタムショップ | 安定性とローンチ | ドラッグは硬め、クルーズは柔らかめ |
| **ビジュアル** | ペイントハブ | 美観 / ロールプレイ | レース仕様にはカーボンファイバー |
| **プレート** | カスタムショップ | パーソナライズ | ロールプレイ用のユニークなID |

<Callout type="tip" title="チューニング戦略">
新しいマップルートをテストするときは、常にバランスの取れたギア比から始めてください。スラカルタへの高速道路のように長い直線があるルートなら最高速度を優先し、短くテクニカルなルートなら加速を優先しましょう。
</Callout>
