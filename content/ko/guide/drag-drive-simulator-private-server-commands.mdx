<Section icon={Settings} title="필수 드래그 드라이브 시뮬레이터 개인 서버 명령어">

관리 권한의 핵심은 채팅 기반의 명령어 시스템에 있습니다. 최근 업데이트에서 도입된 가장 중요한 명령어는 멀티스폰(multispawn) 유틸리티로, 플레이어가 비교나 전시를 위해 맵에 다양한 차량을 생성할 수 있도록 해줍니다.

<CardGrid cols={2}>
  <Card title="차량 관리">
    - **!multispawn me [1-5]**: 현재 탑승 중인 차량을 여러 대 스폰합니다.
    - **!clear**: 스폰된 모든 차량을 제거하여 물리 엔진을 리셋합니다.
    - **!fix**: 차량이 지형에 끼었을 때 현재 차량을 리셋합니다.
  </Card>
  <Card title="서버 관리">
    - **!kick [사용자명]**: 세션에서 방해되는 플레이어를 강퇴합니다.
    - **!ban [사용자명]**: 해당 사용자가 서버에 다시 접속하지 못하도록 영구 차단합니다.
    - **!unban [사용자명]**: 특정 플레이어의 차단을 해제합니다.
  </Card>
</CardGrid>

<Callout type="warning" title="성능 경고">
최대값(5)으로 `!multispawn`을 사용하면 저사양 모바일 기기에서 서버 성능과 프레임 레이트에 큰 영향을 미칠 수 있습니다. 고속 레이스 중에는 주의해서 사용하세요.
</Callout>
