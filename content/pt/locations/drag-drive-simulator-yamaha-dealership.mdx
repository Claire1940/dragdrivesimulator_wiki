<Section icon={Settings} title="Tunando sua Compra para Velocidade Máxima">

Depois de garantir sua moto na **drag drive simulator yamaha dealership**, você deve visitar o menu de tunagem para desbloquear seu verdadeiro potencial. O sistema de relação de marcha é o coração do desempenho neste simulador. Uma configuração de "Relação Curta" proporcionará uma largada violenta, mas atingirá o limitador de rotação muito cedo, enquanto uma configuração de "Relação Longa" pode ter uma velocidade final maior, mas será lenta na linha de partida.

Para motos como a Aerox, uma abordagem equilibrada é a melhor. Você quer torque suficiente para sair da linha sem empinar excessivamente, mas final suficiente para terminar a corrida na Pista Sentul antes que seu oponente o alcance.

| Parâmetro de Tunagem | Efeito de Relação Curta | Efeito de Relação Longa | Recomendado para Yamaha |
| :--- | :--- | :--- | :--- |
| **Transmissão Final** | **Largada mais Rápida** | Maior Velocidade Final | Faixa de 3.5 - 4.2 |
| **1ª Marcha** | Alto Torque | Partida Lenta | Curta para saída rápida |
| **Última Marcha** | Limite de Giro Precoce | **Tração Estendida** | Longa para a linha de chegada |
| **Suspensão** | Mais Rígida (Arrancada) | Mais Macia (Passeio) | Traseira Rígida / Dianteira Macia |

  <Step num="1" title="Corrida de Base Inicial">

  <Step num="2" title="Ajustar Transmissão Final">
